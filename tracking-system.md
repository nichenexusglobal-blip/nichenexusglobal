## 跟进计划

### Day 2 (5/23)
- 检查收件箱，看是否有回复
- 对未回复的，发跟进邮件（不同主题）

### Day 3-4 (5/24-5/25)
- 继续跟进
- 如果有回复 → 进入对话流程
- 如果无回复 → 换联系人（COO/供应链负责人）

### 回复话术矩阵

| 回复 | 应对 |
|------|------|
| "有兴趣" | 立即发送定制化分析报告，约15分钟电话 |
| "发更多信息" | 发案例+数据，不提价格 |
| "现在没时间" | 保留，1个月后跟进 |
| "已经有供应商了" | 提供竞品供应链动向 |
| 无回复 | 3天后换主题再发 |
