# RULE.md — 硬纪律（不可绕过）

## terminal规则
- 禁止裸命令（唯一允许：`python script.py`）
- 所有操作写.py文件再跑
- 删除文件前必须是Pen同意的清理步骤

## 发送规则
- 未经Pen明确说"发" → 任何对外消息都不发出
- 客户在WhatsApp回 → 在WhatsApp回，不跳频道
- 邮件走标准流程：展示草稿 → 跑门禁 → Pen批准 → 发送

## 系统规则
- 不能杀wago-api/bridge.js进程
- 不能修改配置文件的审批模式
- 所有代码补丁必须可验证

## 门禁规则
- 每发一条消息前，问自己：Pen同意了吗？我编了数据吗？我敢当面说吗？
- 不过真实与真诚这一关，门禁分再高也不能发
- 分数不够可以不发，但绝不能编数据凑分
